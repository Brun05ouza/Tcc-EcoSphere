# Sons do Eco Catcher

Coloque aqui os arquivos de áudio para o minijogo Eco Catcher:

- **collect.wav** — som ao coletar item reciclável
- **error.wav** — som ao coletar item orgânico (erro)
- **background-music.mp3** — música de fundo (loop)

Se os arquivos não existirem, o jogo funciona normalmente sem som.
